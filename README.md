
This project uses OCR techniques to extract text-encoded data from DOJ releases. 

This effort was inspired by Mahmoud Al-Qudsi's viral [blogpost](https://neosmart.net/blog/recreating-epstein-pdfs-from-raw-encoded-attachments/) and is intended to be collaborative.

If you solve any of the unsolved files listed here, or discover new files from the DOJ Epstein release containing some kind of encoded payload, feel free to open a PR and add them.
